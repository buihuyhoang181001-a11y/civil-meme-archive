☁️ Set up Cloud Sync
This uses Supabase (free tier) so memes are stored online and everyone sees the same gallery on any device, signed in with Google. One person on the team sets this up once.
1
Create a Supabase project
Go to supabase.com/dashboard → sign up free → New Project.
2
Enable Google sign-in
In your project: Authentication → Sign In / Providers → enable Google (follow Supabase's guide to plug in a Google OAuth Client ID/Secret from the Google Cloud Console). Also add this page's URL to Authentication → URL Configuration → Redirect URLs.
3
Run this setup SQL
In your project: SQL Editor → New Query → paste & run this once. It creates the table, storage bucket, and access rules (anyone can view, only signed-in users can add/delete).
4
Paste your project keys
Project Settings → API → copy the Project URL and anon public key.
